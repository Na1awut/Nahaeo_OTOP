# 📁 โมเดล 3D - ต้นดอกไม้ (ประเพณีแห่ต้นดอกไม้)

## รายละเอียด
- **ชื่อ:** ต้นดอกไม้ / ประเพณีแห่ต้นดอกไม้ยักษ์
- **ที่ตั้ง:** อำเภอนาแห้ว จังหวัดเลย
- **ความสำคัญ:** ประเพณีเก่าแก่ 400 ปี บุญแห่ต้นดอกไม้

## ไฟล์ที่ต้องวาง
```
ton-dok-mai/
├── model.glb         # โมเดลหลักสำหรับ Web
├── model.usdz        # โมเดลสำหรับ iOS AR Quick Look
├── thumbnail.jpg     # รูปตัวอย่าง (512x512)
└── textures/         # (ถ้ามี) texture แยก
```

## ขนาดแนะนำ
- **Polygon count:** < 100,000 triangles
- **Texture resolution:** 2048x2048 หรือต่ำกว่า
- **File size:** < 5MB (สำหรับ mobile)
